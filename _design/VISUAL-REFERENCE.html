Card shells
All gallery cards are 300×280px. No exceptions. Four shells. Every component type maps to one shell.
Universal rules: 300px width · 280px height · 7px solid teal border all sides · white interior · source stamp bottom-right · all cards in one carousel same height

Interactive element placement
Three valid slots for interactive tools (sliders, calculators, compare tools, dropdowns). Piece brief specifies which slot.
SlotDescriptionExample
Slot 1 — Tab primaryInteractive tool IS the tab content. No card gallery on this tab.Sodium "Compare Your Order" tab
Slot 2 — Below galleryCard gallery first, then interactive tool below it on same tab.Concert Tax "The Receipt" on Data tab
Slot 3 — Above galleryInteractive tool as lead-in, card gallery below on same tab.Not yet used — available for future pieces
Rules: Interactive tools use white/paper backgrounds (never dark). 44px min touch targets. Piece-local CSS with prefix. Tool must have a visible source citation. One interactive per tab max.
