Locked Rules — Applied to Every Component
Bar Scaling
Longest bar = 90% of track width. All bars = (value / max_value) × 90. Never hardcode from raw data percentages. Max empty track: 10% on longest bar only.
Value Alignment
All bar values right-aligned to the right edge of the full track. Never inside the colored bar. Full ink always (#111 on white, #fff on teal field). No value moves horizontally based on bar length.
Primary Label Scale
Franchise names, decade labels, city names, subject names are headlines. Minimum 24px, target 28–32px, Spectral 800. Size the column to fit the label — never shrink the font to fit the column.
Label Column + Gap
Label column: 40% of card. Minimum 20px gap between label and track always. If longest label wraps, expand column and compress track. Never reduce font size.
Benchmark Labels
Benchmark labels (e.g. "Daily limit: 1,500mg") match bar values in size and weight: DM Mono 700, 14px minimum.
Color as Field
Color owns the entire surface. The card IS the color. Never use color as a stripe or accent when the argument warrants full commitment. Tonal depth within a single hue — no flat fills.
Surface Options
Every component has two options: Option 1 (teal field #0a7c72, white as relief) and Option 2 (white #fff, 7px teal left border). AMA locks one per component per piece before deploy.
No Gray in Content Hierarchy
Gray (#888) is permitted only on source stamps and comp-tag labels. Never on primary labels, values, editorial lines, or annotations.
Empty Space Cap
No more than 5–10% empty space in any component excluding left/right padding. If space is empty, either scale up the bars, increase the font, or widen the label column.
