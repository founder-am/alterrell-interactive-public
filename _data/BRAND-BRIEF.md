## COLOR SYSTEM — TABLED

A full color rebrand was explored and tabled. Reason: pieces are actively shipping. A mid-calendar rebrand introduces risk without editorial payoff. The existing color system remains in place. Color rebrand is deferred until after Congress Part 1A ships.

**What was explored and rejected:**
- Flat accent swaps (Deep Crimson, Forest, Burgundy) — rejected, insufficient depth
- Full dark-field system (navy as primary surface) — directionally right but wrong timing
- theme-v2.css — file not created, session closed before any code was written

**What was confirmed as directionally correct for a future rebrand session:**
- Navy as field color, not accent — fills containers completely, white as relief
- Tonal depth within a single hue (reference: Delta, AmEx, Starbucks, WB)
- White earned through contrast, not defaulted as negative space
- Shareable assets tab per piece eliminates mobile screenshot constraints on site design

**Standing token system remains:**
- `--navy: #0F172A` — nav, active tabs, buttons, stat bands, hub card border
- `--electric-blue: #2563EB` — "Interactive" wordmark only
- `--white: #FFFFFF` — all content backgrounds
- `--ink: #0F172A` — all body text and headlines
- `--muted: #64748B` — italic aside text beside CTA buttons only
- `--border: #E2E0DC` — section dividers, card borders
- `--highlight: TBD` — deferred to post-Congress rebrand session
- Teal `#0a7c72` — retired
- Paper `#f8f6f1` — retired
- Orange `#c95f1a` — retired

## OBSIDIAN FUTURES — STATUS UPDATED

Obsidian Futures is no longer a content lane. It is a label applied to individual pieces.

All pieces are Alterrell Interactive. The Obsidian Futures label signals content mode: cultural, celebratory, community-forward, educational. It does not imply a separate design system or publication.

The dark treatment (`#0a0a12` bg, `#E8B923` gold accent) used in existing Obsidian Futures pieces is retained for those pieces. It does not extend to new pieces unless the content mode warrants it and AMA explicitly approves it per piece.
