# Alterrell Interactive — Editorial System Framework

## Purpose

This framework is designed to turn:
- essays
- voice notes
- transcripts
- fragmented ideas
- cultural observations
- research dumps
- Twitter threads
- partially formed arguments

into:
- premium editorial interactives
- mobile-first think pieces
- visual card systems
- tabbed reading experiences
- Substack companion experiences
- data-driven cultural essays

This is NOT a generic content workflow.
The goal is to preserve:
- intellectual sharpness
- sociological framing
- emotional undercurrent
- cultural specificity
- argumentative structure
- visual editorial cohesion

while translating longform thinking into a modular interactive experience.

The output should feel like:
- Financial Times visual essays
- Vox explanatory interactives
- Museum exhibit placards
- Luxury editorial design
- Cultural criticism with data storytelling

NOT like:
- Instagram activism
- corporate DEI decks
- generic thinkpieces
- YouTube explainer sludge
- infographic spam

---

# MASTER PROCESS

The process has 7 phases.

1. Raw Thought Dump
2. Argument Extraction
3. Structural Compression
4. Tab Architecture
5. Card System Design
6. Voice Pass Writing
7. Mobile Experience Calibration

---

# PHASE 1 — RAW THOUGHT DUMP

## What To Paste Into ChatGPT/Claude/Gemini

Paste:
- essays
- transcripts
- rough notes
- screenshots of threads
- bullet points
- half-written observations
- research excerpts
- comparisons
- examples
- quotes
- frustrations
- voice note transcripts

Do NOT over-organize beforehand.
The system works best when the intellectual raw material still contains emotional energy and unfinished connective tissue.

The goal is NOT polished writing.
The goal is pattern density.

---

# PHASE 2 — ARGUMENT EXTRACTION

## Prompt

Read everything below and identify:

1. The central thesis
2. The hidden sociological argument
3. The emotional engine beneath the argument
4. The repeated patterns/examples
5. The strongest comparative contrasts
6. The systems-level observations
7. Which ideas are:
   - emotional
   - economic
   - cultural
   - historical
   - racial
   - gendered
   - structural
8. What the audience is supposed to “unsee” once they understand the argument
9. Which ideas deserve:
   - prose
   - charts
   - timelines
   - pull quotes
   - comparison cards
   - diagrams
10. The recurring phrases or language that feel foundational to the piece

Do NOT summarize.
Instead extract the intellectual architecture.

---

# PHASE 3 — STRUCTURAL COMPRESSION

## Purpose

Convert the longform argument into a mobile-native experience.

The system should:
- reduce cognitive overload
- preserve complexity
- maintain emotional escalation
- separate ideas into digestible visual modules

The interactive should feel:
- swipeable
- layered
- scroll-native
- argument-driven
- visually paced

NOT like reading an essay pasted onto a phone.

---

# PHASE 4 — TAB ARCHITECTURE

## Prompt

Based on the material provided, create:

1. A mobile-first tab structure
2. The ideal progression of ideas
3. The emotional pacing of the experience
4. Which sections should feel:
   - analytical
   - historical
   - emotionally sharp
   - data-heavy
   - reflective
   - confrontational
5. Which visuals belong inside each tab
6. Which arguments deserve their own visual emphasis

Each tab should contain:
- 2–4 short paragraphs maximum
- several editorial cards
- one core insight
- one emotional takeaway
- one visual thesis

Tabs should feel like chapters in a museum exhibit.

The tabs should NOT repeat the same point.
Each tab should advance the argument.

---

# PHASE 5 — CARD SYSTEM DESIGN

## Purpose

Cards are the core storytelling engine.

Every card should:
- isolate one idea
- simplify comparison
- create visual contrast
- reward quick reading
- support screenshots/sharing
- preserve intellectual density

---

# Core Card Types

## 1. Contrast Card

Use when comparing:
- men vs women
- labor vs reward
- effort vs compensation
- production vs minimalism
- perception vs reality

Structure:
- split vertically or horizontally
- asymmetry is encouraged
- one side should visually feel “lighter”
- the other side should feel denser/more labor intensive

---

## 2. Deficit / Financial Card

Use for:
- spending gaps
- labor imbalance
- production economics
- hidden costs

Should feel:
- financial
- severe
- clean
- impossible to ignore

Use oversized typography for:
- deficits
- percentages
- labor counts
- disparities

---

## 3. Timeline Card

Use when:
- internet discourse reshaped someone
- expectations escalated over time
- criticism altered career trajectory

The visual should communicate:
- transformation
- escalation
- pressure
- narrative progression

---

## 4. Double Standard Card

Use:
- checkmarks vs penalties
- forgiveness vs punishment
- acceptable vs unacceptable behavior

These cards work best when:
- one column feels casual
- the other feels exhausting

---

## 5. Pull Quote Card

Use to:
- punctuate sections
- slow pacing
- create screenshot moments
- reinforce thesis

Rules:
- short
- sharp
- sociological
- emotionally sticky
- visually elegant

---

# PHASE 6 — VOICE PASS WRITING

## Purpose

The prose between cards should:
- connect ideas
- escalate argument
- interpret visuals
- provide sociological framing

It should NOT:
- over-explain the cards
- repeat statistics excessively
- become academically dense
- become Tumblr-style activism

The prose voice should feel:
- intelligent
- emotionally restrained
- culturally literate
- observant
- confident
- provocative without yelling

Sentence structure should alternate between:
- elegant longform observation
- short declarative statements

---

# PHASE 7 — MOBILE EXPERIENCE CALIBRATION

## Important Rules

Mobile users fatigue quickly.

The experience should:
- create rhythm
- alternate density levels
- mix prose with visual breaks
- avoid giant walls of text
- maintain momentum

A good mobile editorial interactive feels like:
- controlled revelation
- progressive argument layering
- visualized thinking

NOT information dumping.

# OPTIONAL ENHANCEMENTS

## If Using Voice Notes

Prompt:

Treat the transcript as intellectual raw material, not polished speech.
Extract:
- recurring emotional tension
- repeated metaphors
- instinctive contrasts
- moments where the speaker becomes sharper or more animated
- latent sociological framing

Do NOT over-clean the voice.
Preserve the thinking style.

---

## If Using Research Dumps

Prompt:

Do not summarize the research.
Instead identify:
- which facts create visual tension
- which statistics support emotional arguments
- which examples repeat the same structural pattern
- what the audience will find hardest to ignore
