# Naming Data

Reference files for the Obsidian Futures "What's in a Name" series.

## Contents
Place the following files from the naming data zip in this folder:
- All .md files (tier lists, analysis, scripts, episode briefs)
- All .html files (built charts)

## Usage
These files are tracked in git for Claude Code reference.
Raw SSA data files (.txt, .csv) are gitignored — do not commit them here.

## Key files
- tier1_master_list_COMPLETE.md — canonical name tier rankings
- sitcom_names_analysis.md — sitcom character/actor crossreference
- sitcom_actor_crossreference.md — full cast analysis by show
- complete_tier_system.md — all three buckets with data
- episode_3_sitcoms_script.md — Part 2 episode script draft
- living_single_indexed.html — Living Single chart (built)
- whitney_houston.html — Whitney chart (built)
- mariah_final.html — Mariah chart (built)
