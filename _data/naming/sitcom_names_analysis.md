# Ethnic & Racialized Names from Black Sitcoms (1960-2019 Analysis Period)

## Priority: Sitcom Character Names

### Living Single (1993-1998)
**Show Context:** Fox series about six Black friends in Brooklyn. Peak cultural moment for Black sitcoms.

- **Khadijah** (Khadijah James) - Arabic origin, Muslim name
- **Regine** (Regine Hunter) - French, but uncommon in mainstream US
- **Sinclaire** (Sinclair James) - Surname-as-first-name, upscale/aspirational
- **Kyle** (Kyle Barker) - More common but included
- **Overton** (Overton Wakefield Jones) - Distinctive surname-as-first-name

### A Different World (1987-1993)
**Show Context:** Cosby Show spinoff, set at fictional HBCU Hillman College.

- **Whitley** (Whitley Gilbert) - Southern aristocratic name, unexpected for Black character
- **Dwayne** (Dwayne Wayne) - More mainstream, but significant in AAVE context  
- **Jaleesa** (Jaleesa Vinson-Taylor) - African American coinage, distinctive

### Fresh Prince of Bel-Air (1990-1996)
**Show Context:** Will Smith's breakout, juxtaposing West Philly with Bel-Air wealth.

- **Carlton** (Carlton Banks) - Preppy, upper-class signifier
- **Hilary/Hillary** (Hilary Banks) - Traditional but became associated with character's persona

### Martin (1992-1997)
**Show Context:** Martin Lawrence sitcom, working-class Detroit setting.

- **Gina** (Gina Waters-Payne) - Short form that became standalone
- **Martin** (dual: Martin Lawrence character + MLK reference)

## Analysis Recommendations

### Tier 1: Highest Priority (Most "Ethnic" Signal)
These names are most likely to show the "ghetto until famous" pattern:

1. **Khadijah** - Explicitly Muslim/Arabic, highly racialized
2. **Regine** - Uncommon, foreign-sounding
3. **Whitley** - Unexpected for Black character, upper-class coding
4. **Jaleesa** - Clear African American linguistic innovation
5. **Sinclaire** - Upscale surname-as-first usage
6. **Moesha** - Distinct contemporary Black coinage
7. **Akeelah** - Unique spelling signals cultural specificity

### Tier 2: Moderate Ethnic Coding
Names that read as "different" but not overtly racialized:

1. **Carlton** - Preppy/upper-class coded
2. **Overton** - Unusual surname-as-first
3. **Dwayne** - AAVE associations
4. **Tiana** - Exotic but not specifically racialized
5. **Antwone** - Spelling variation signals difference

### Tier 3: Context-Dependent
These need show debut timing to determine impact:

1. **Kyle**, **Marcus**, **Gina** - More common names where show context matters
2. **Hilary** - Traditional name but character-specific association
