## PRODUCTION NOTES

**Pacing:**
- Fast cuts (1-2 seconds per image)
- Punchy stats appearing on screen
- No dwelling - create urgency
- Music drives momentum

**Music:**
- Start minimal (just bass/drums)
- Build intensity with each stat
- Drop during "This is about when America decides"
- Crescendo into series overview
- Cut to silence for final branding

**Text Animation:**
- Stats should POP onto screen
- Bold, yellow (#E8B923) on black
- Large enough to read on mobile
- Chart snippets should animate (line drawing effect)

**B-Roll Sources:**
- All episode chart snippets
- Celebrity footage (licensed)
- Historical civil rights footage
- Modern Black culture
- Hospital nurseries
- Resume screening studies

**Call to Action:**
- "Subscribe now" button appears bottom third
- Countdown timer optional ("Series begins in X days")
- Social media handles in lower corner

**Versions:**
- 90-second version (YouTube pre-roll)
- 60-second version (Instagram/TikTok)
- 30-second version (Instagram Stories)
- 15-second version (Twitter/X)

## A/B TESTING VARIANTS

**Variant A - Shock Value:**
Start with: "Jamal is ghetto. Mariah is ghetto. Aaliyah is ghetto."
Then: "Until they're not."

**Variant B - Question Hook:**
Start with: "Why do names go from 'ethnic' to 'popular' overnight?"
Then: "I analyzed 60 years of data to find out."

**Variant C - Personal Story:**
Start with: "Someone once told me my name sounded 'too ethnic' for corporate America."
Then: "So I analyzed when America decides names are acceptable."

Test all three, measure:
- Click-through rate
- Watch time
- Conversion to subscribe
